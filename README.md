# OpenADMET PXR Challenge Model Report

This repository contains a preliminary model report for a Track 1 Activity
submission to the OpenADMET PXR Blind Challenge.

The report is intentionally concise while the challenge is ongoing. A more
detailed technical write-up may be released after the competition concludes.

- Report: [MODEL_REPORT.md](MODEL_REPORT.md)
- Track: Track 1 Activity
- Task: blinded pEC50 prediction for PXR compounds

No prediction files, private experiment logs, model checkpoints, or competition
runtime artifacts are included in this public repository.
